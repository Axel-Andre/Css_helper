# Css helper tool

A tool that allows you to visualize CSS properties to better understand the syntax. I'm building this project to learn Vue.js and his different tools, and also to sharpen my skills in CSS.

## Built With

- [Vue.js](https://vuejs.org/) - The web framework used
- [Vue CLI 3](https://cli.vuejs.org/) - Standard Tooling for Vue.js Development
- [Vuex](https://vuex.vuejs.org/) - Vuex is a state management pattern + library for Vue.js applications.
